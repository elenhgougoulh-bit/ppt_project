 🪄 AI PowerPoint Generator

Ένα αυτοματοποιημένο εργαλείο που δημιουργεί επαγγελματικές παρουσιάσεις PowerPoint (.pptx) σε δευτερόλεπτα, χρησιμοποιώντας Τεχνητή Νοημοσύνη (Google Gemini) και δυναμική άντληση εικόνων (Unsplash API). Το project διαθέτει ένα φιλικό Web UI φτιαγμένο με Streamlit.

## 🚀 Χαρακτηριστικά
* **AI Content Generation:** Παραγωγή δομημένου κειμένου (τίτλοι, bullets, quotes) με το Google Gemini 2.5 Flash.
* **Auto-Image Sourcing:** Αυτόματη εύρεση και εισαγωγή σχετικών εικόνων υψηλής ανάλυσης μέσω του Unsplash API.
* **Custom Templates:** Λειτουργεί με οποιοδήποτε προφτιαγμένο PowerPoint Template (Slide Master), διατηρώντας το εταιρικό σας design ανέπαφο.
* **Web Interface:** Εύχρηστο περιβάλλον χρήστη με το Streamlit.
* **Dockerized:** Πλήρως στημένο σε DevContainer για άμεση εκτέλεση χωρίς προβλήματα με dependencies.

## 🛠️ Τεχνολογίες που χρησιμοποιήθηκαν
* Python 3.11
* [python-pptx](https://python-pptx.readthedocs.io/) (Διαχείριση παρουσιάσεων)
* [google-genai](https://pypi.org/project/google-genai/) (Google Gemini API)
* Unsplash API
* Streamlit (Web UI)
* Docker / DevContainers
